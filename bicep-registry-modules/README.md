# bicep-registry-modules

This folder is a placeholder for Bicep registry modules.

## Purpose

This folder will be automatically populated by the GitHub Actions workflow
that publishes Bicep modules to your Azure Container Registry.

## Usage

Do not manually add files to this folder. The workflow will:
1. Build your Bicep modules
2. Publish them to your Azure Container Registry
3. Update this folder with the module references

---
*This folder was created by Azure Deployment Framework*
